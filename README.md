# Termix
A chatting client based on Socket.h in Linux terminal
___
## Terminal Commands
`cd Termix`
`gcc server.c -lpthread`
`./a.out`

**(ON NEW TERMINAL)**
`cd Termix`
`gcc client.c -lpthread`
`./a.out user1 hostIp 9999`

**(ON NEW TERMINAL)**
`cd Termix`
`gcc client.c -lpthread`
`./a.out user2 hostIp 9999`
